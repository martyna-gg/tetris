# Tetris

Tetris game with usage of random module, numpy array, object oriented programming, if/else/elif statements, while and for loops, dicts, map and lambda functions, reading files and processing user input.

## Installation

Random module is a python built-in module. No installation is required.

Numpy module has to be installed using pip:
```
$ pip3 install numpy
```

## Requirements

Code was developed and tested using: 

Ubuntu 18.04.6 LTS 

Python 3.6.9

## Usage

The game starts with menu with three options: play a game, see the best results or exit.

```
$ python3 tetris.py
select g for game, b for best score or e for exit >
```
When the best score option is selected the best result is printed and player comes back to the previous menu.
```

select g for game, b for best score or e for exit >
```
